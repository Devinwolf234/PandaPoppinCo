# Panda Poppin Co - E-Commerce Concept Site

A concept e-commerce storefront built for a hunting 
YouTube channel. Designed as a functional prototype 
to explore web development and real-world application.

## Features
- Product catalog with hoodies, hats, and t-shirts 
  in multiple color variants
- Shopping cart functionality
- Contact form
- Google Sheets integration for order submission

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Sheets API

## Notes
Built as a personal project to explore front-end 
development and e-commerce functionality.
